## 作品评选
评委会根据「代码质量」「视觉设计」「用户体验」「作品完成度」「创意度」「潜在商业价值」等多个维度进行考量。

*使用特别版SDK的团队，由于开发难度加大，因此会酌情加分*

* **完成度（40%）**
参赛团队提供的作品实现程度。
* **创意度（20%）**
参赛作品的创新程度。是否具备充分的创新性，与行业内同类产品具有差异性。
* **实用性（20%）**
参赛作品在应用场景中的实际应用程度。是否能够匹配用户的发展诉求，注重用户体验。
* **潜在商业价值（20%）**
参赛作品可以在日常工作生活中产生的商业价值。是否能够引起行业关注和足够的影响力以及创造市场效益。
